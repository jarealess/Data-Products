App description

This aplication can be used to study the relationship between the speed and the distance reached by a car. The data used was taken from the CARS data set. You can fit the regression lines, show the leats squared lines and see a prediction made by the model for a given speed value and its dist value over the red line. The application also shows the diagnostics plots, where the user can select a number from 1 to 6 and see the respective diagnostic plot he/she wants to explore.
